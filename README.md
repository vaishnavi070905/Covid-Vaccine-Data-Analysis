
# 🦠 COVID-19 Vaccine Data Analysis

## 📊 DSBDA Mini Project

## 📌 Project Overview
This project focuses on analyzing COVID-19 vaccination data in India using Python. The analysis includes state-wise vaccination distribution, comparison of first and second doses, and gender-wise vaccination trends. The goal is to extract meaningful insights using data analysis and visualization techniques.

---

## 🎯 Objectives
- To analyze COVID-19 vaccination data  
- To study state-wise vaccination distribution  
- To compare first dose and second dose coverage  
- To analyze male and female vaccination  
- To visualize data using charts  

---

## 📂 Dataset Description
The dataset contains vaccination details of different states and union territories of India.

### Attributes:
- **State** – Name of the state/UT  
- **First_Dose** – Number of people vaccinated with first dose  
- **Second_Dose** – Number of people vaccinated with second dose  
- **Male** – Number of males vaccinated  
- **Female** – Number of females vaccinated  
- **Total_Doses** – Total vaccination doses  

---

## 🛠️ Tools & Technologies
- Python  
- Pandas  
- Matplotlib  
- Seaborn  
- Google Colab  

---

## ⚙️ Implementation Steps
1. Load dataset using Pandas  
2. Explore dataset using `head()` and `describe()`  
3. Perform state-wise analysis  
4. Calculate total male and female vaccination  
5. Visualize data using bar chart and pie chart  

---

## 📊 Visualizations
- 📊 Bar Chart – State-wise total vaccination  
- 🥧 Pie Chart – First dose vs Second dose  

---

## 📈 Key Insights
- Vaccination varies across states  
- Highly populated states have higher vaccination counts  
- First dose vaccination is higher than second dose  
- Male and female vaccination is relatively balanced
